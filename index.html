Die Roller!

Enter the number of dice + or - pips and press enter. Example: 7+2 or 7-2 means 7D6 + 2

Some other commands:
  • * or = will clear the results table
  • - will redo the last roll with one less die
  • Enter will redo the last roll exactly
  • c will add a character point to the last roll.
